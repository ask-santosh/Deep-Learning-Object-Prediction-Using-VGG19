# Deep-Learning-Object-Prediction-Using-VGG19

### Introduction
VGG19 model for object prediction .

### Run Application
1. Run model_generator.ipynb file for generating VGG19 model . 
2. python app.py
3. run with 127.0.0.1:5000 port number .
